# P04 收藏夹详情页 产品需求文档

> 版本: v1.0 | 日期: 2026-03-29 | 状态: 待开发 | 父模块: M14-收藏系统
> 路由: /collections/:id (Web) | /collections/[id] (Mobile) | 含分享页 /collections/shared/:token

---

## 1. 需求背景与目标

收藏夹详情页展示单个收藏夹的全部内容项，支持管理操作（移除、备注、移动）以及内容分享。分享页（/collections/shared/:token）是同一页面的只读公开版本，匿名可访问，供用户将收藏夹分享给朋友或同行者。

**页面目标：**
- 用户能清晰浏览收藏夹中的所有内容，支持多类型内容卡片（圣地/祖庭/祖师）
- 用户能高效管理收藏项（删除/备注/移动到其他收藏夹）
- 分享页面提供优质只读体验，激励访客注册并收藏

---

## 2. 用户故事

### US-01 浏览收藏夹内容

**作为** 有朝圣计划的用户，
**我希望** 在收藏夹详情页看到所有收藏内容的图片和关键信息，
**以便** 快速回顾收藏的内容并决定加入哪些到行程。

**验收标准：**
- [ ] 页面顶部展示封面大图（高度 240px）+ 收藏夹名称 + 描述 + 项目总数
- [ ] 收藏项以内容卡片列表展示，圣地/祖庭/祖师各用对应卡片样式
- [ ] 每张卡片显示：内容图片、名称、所属信仰/类型、用户备注（若有）
- [ ] 默认按添加时间倒序排列，支持「按类型」「自定义」排序切换
- [ ] 加载更多采用分页（每页 20 项），滚动到底部自动加载

### US-02 移除收藏项并添加备注

**作为** 整理收藏夹的用户，
**我希望** 对收藏项进行删除和备注操作，
**以便** 记录我对每个圣地的个人想法，保持收藏夹整洁。

**验收标准：**
- [ ] 每个收藏项卡片右上角有「...」操作按钮（Web）或左滑操作（Mobile）
- [ ] 操作菜单包含：编辑备注 / 移动到其他收藏夹 / 移除
- [ ] 编辑备注弹窗：文本域，最多 200 字，支持清空
- [ ] 移除操作需二次确认（「确认从收藏夹移除？」）
- [ ] 操作成功后卡片动画移出，列表重排

### US-03 分享收藏夹给他人

**作为** 朝圣旅行领队，
**我希望** 生成分享链接发给团队成员，
**以便** 成员可以浏览我精选的圣地列表。

**验收标准：**
- [ ] 页面顶部有「分享」按钮（含分享图标）
- [ ] 点击后生成唯一 shareToken，弹窗显示可复制的链接
- [ ] 分享页（/collections/shared/:token）：无需登录可访问
- [ ] 分享页顶部显示「由 [用户昵称] 整理」+ 收藏夹名称和描述
- [ ] 分享页所有操作按钮（删除/备注/移动）均隐藏，内容为只读
- [ ] 分享页底部有「登录 JOINUS 创建我的收藏夹」注册引导 CTA

### US-04 批量操作收藏项

**作为** 需要整理大量收藏的用户，
**我希望** 多选收藏项后批量删除或移动到其他收藏夹，
**以便** 节省逐个操作的时间。

**验收标准：**
- [ ] 页面右上角有「多选」按钮，点击进入多选模式
- [ ] 多选模式下每个卡片左上角出现复选框，点击选中（选中状态有高亮边框）
- [ ] 底部浮动操作栏：显示已选 N 项 + [批量删除] [批量移动] [取消]
- [ ] 批量删除需二次确认，批量移动弹出收藏夹选择器
- [ ] 操作完成后退出多选模式，列表刷新

## 4. 内容卡片规范

不同 entityType 的卡片样式：

| entityType | 左图内容 | 标题下方标签 | 右侧角标颜色 |
|------------|---------|-------------|-------------|
| HOLY_SITE | 圣地图片 | 「圣地 · {信仰名}」 | 信仰主题色 |
| TEMPLE | 祖庭图片 | 「祖庭 · {信仰名}」 | 信仰主题色 |
| PATRIARCH | 祖师图片 | 「祖师 · {信仰名}」 | 信仰主题色 |
| TRIP | 行程封面 | 「行程 · {状态}」 | 状态对应色 |

卡片点击跳转规则：
- HOLY_SITE → /holy-sites/:entityId
- TEMPLE → /temples/:entityId
- PATRIARCH → /patriarchs/:entityId
- TRIP → /trips/:entityId

---

## 5. 排序方式

| 排序选项 | 说明 |
|----------|------|
| 添加时间（默认）| 按 CollectionItem.createdAt 倒序 |
| 按类型 | 先圣地，后祖庭，后祖师，最后行程；同类按名称 |
| 自定义 | 拖拽排序，修改 CollectionItem.sortOrder，PATCH 接口保存 |

---

## 6. API 调用

| 操作 | API | 说明 |
|------|-----|------|
| 获取详情 | GET /api/collections/:id | 含 items（分页） |
| 获取分享页 | GET /api/collections/shared/:token | 公开，无需登录 |
| 更新收藏夹 | PATCH /api/collections/:id | 名称/描述/封面/公开 |
| 添加备注 | PATCH /api/collections/:id/items/:itemId | 更新 note 字段 |
| 移除收藏项 | DELETE /api/collections/:id/items/:itemId | 单项移除 |
| 生成分享链接 | POST /api/collections/:id/share | 返回 { shareToken, url } |
| 更新排序 | PATCH /api/collections/:id | 更新 items sortOrder 数组 |
| 批量删除 | DELETE /api/collections/:id/items（body: itemIds[]） | 批量移除 |
| 移动到其他夹 | POST /api/collections/:targetId/items（batch） | 跨夹移动 |

## 7. 验收标准

| 标准编号 | 描述 |
|----------|------|
| AC-01 | 详情页首屏（前 20 项）加载 < 2s (P95) |
| AC-02 | 分享页（/shared/:token）无需登录可访问，不显示任何管理操作按钮 |
| AC-03 | 分享页备注字段对外隐藏（不展示用户私人备注） |
| AC-04 | 多选模式下底部操作栏固定在屏幕底部，不随列表滚动 |
| AC-05 | 不同 entityType 的内容卡片样式正确区分，点击跳转到对应详情页 |
| AC-06 | 移除收藏项有二次确认弹窗，取消操作不触发 API |
| AC-07 | 批量删除操作有二次确认，成功后退出多选模式，列表正确刷新 |
| AC-08 | 分享链接弹窗支持一键复制，复制成功有 Toast 反馈 |
| AC-09 | 空收藏夹状态展示引导文案「尚未收藏任何内容」+ 「去浏览圣地」按钮 |
| AC-10 | 自定义拖拽排序保存成功，刷新页面后顺序持久化 |
