# F37.01-PRD-七境界引擎算法

> 圆满之路核心算法 | 输入多维度行为, 输出当前境界 + 晋阶建议

## 1. 职责
根据用户的 M36 评分 + 打卡数据 + 日志 + 问答 + 时长等综合信号, 计算当前境界 (1-7) 和置信度.

## 2. 输入信号 (8维度)

| # | 信号 | 来源 | 权重 |
|---|------|------|------|
| 1 | M36 信仰力评分 | FaithAssessmentResult | 0.20 |
| 2 | 每日印连击天数 | DailySealPractice | 0.15 |
| 3 | 累计修行时长 (min) | DailySealPractice.audioListenedSec | 0.10 |
| 4 | 因缘日志数 + AI平均境界标注 | KarmaEvent | 0.15 |
| 5 | 融通问答次数 + 深度分数 | WisdomQuery | 0.10 |
| 6 | 十牛图当前阶段 | FulfillmentJourney.oxStage | 0.15 |
| 7 | 朝圣日志中的真实圣地到访 | Journal | 0.10 |
| 8 | 师徒传承角色 (弟子/导师) | MasterDiscipleLineage | 0.05 |

## 3. 算法公式

```typescript
function calcRealm(signals: Signals): { realm: Realm, confidence: number } {
  // 1. 每个信号标准化到 [0, 7]
  const s1 = normalize(signals.m36Score, 0, 100, 0, 7);
  const s2 = normalize(signals.streakDays, 0, 365, 0, 7);
  const s3 = normalize(signals.totalPracticeMin, 0, 10000, 0, 7);
  const s4 = signals.avgKarmaRealm ?? 0;
  const s5 = normalize(signals.wisdomDepth, 0, 50, 0, 7);
  const s6 = normalize(signals.oxStage, 1, 10, 1, 7);
  const s7 = normalize(signals.holySitesVisited, 0, 30, 0, 7);
  const s8 = signals.lineageRole === 'MASTER' ? 7 : signals.lineageRole === 'DISCIPLE' ? 4 : 2;

  // 2. 加权求和
  const raw = 0.20*s1 + 0.15*s2 + 0.10*s3 + 0.15*s4
            + 0.10*s5 + 0.15*s6 + 0.10*s7 + 0.05*s8;

  // 3. 离散化为 1-7
  const realm = Math.max(1, Math.min(7, Math.round(raw)));

  // 4. 置信度 = 信号完整度 × 方差反比
  const presence = countNonZero(signals) / 8;
  const variance = calcVariance([s1,s2,s3,s4,s5,s6,s7,s8]);
  const confidence = presence * (1 / (1 + variance));

  return { realm, confidence };
}
```

## 4. 晋阶规则
- 新境界 > 当前境界 AND 置信度 ≥ 0.7 AND 稳定 7 天 → 允许晋阶
- 新境界 < 当前境界 → 不降级, 只提示"建议回归基础"
- 置信度 < 0.5 → 显示"境界待定"

## 5. 频率
- 每日凌晨 CRON 批量重算
- 用户触发关键动作 (完成印/日志/问答) 时实时重算
- 每周日快照写入 realmScoreSnap

## 6. 单元测试
- normalize 边界
- 加权求和正确性
- 置信度计算
- 稳定天数判定
- 全零输入 → 初觉 AWAKENING
- 全满输入 → 布施 GIVING_BACK
