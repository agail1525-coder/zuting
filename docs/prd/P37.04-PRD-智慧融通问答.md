# P37.04-PRD-智慧融通问答

> 圆满之路 A4 轴 | 路由 /trips/cultivation/synthesis | 全球独创 12祖师同台回答

## 1. 页面定位
用户提出一个人生困惑, AI 同时以 12 位文化祖师的视角回答, 用户选择或融合形成"我的答案".

## 2. 产品核心创新
- **同台回答**: 单次提问, 12 路并行 LLM 调用 (小鸿 Qwen3.5-35B), 流式输出
- **祖师人设**: 惠能/老子/耶稣/穆罕默德/商羯罗/摩西/孔子/那纳克/天照/莲花生/部落长老/巴哈欧拉
- **融合引擎**: 用户选择 2-3 个答案后, 再调用一次 LLM 生成"你的融合答案"
- **引注**: 每个答案必须含原典引文 (经文/公案/圣训)

## 3. 布局
```
┌────────────────────────────────────────┐
│ 你的困惑:                              │
│ [我该原谅背叛我的合伙人吗?          ] │
│ [                       提问→         ]│
├────────────────────────────────────────┤
│ 禅宗·惠能: "不思善不思恶..."  ⭐选   │
│ 基督·耶稣: "七十个七次..."    ⭐选   │
│ 道家·老子: "报怨以德..."             │
│ 伊斯兰·穆圣: "忍耐与宽恕..."         │
│ ... (12路流式)                        │
├────────────────────────────────────────┤
│ 已选 2 → [生成融合答案]                │
│                                          │
│ 你的融合答案: "原谅不是为他..."         │
│ [保存到因缘日志]                        │
└────────────────────────────────────────┘
```

## 4. API
- `POST /trips/cultivation/wisdom/query` body: `{ question }` → SSE 流式返回 12 路答案
- `POST /trips/cultivation/wisdom/synthesize` body: `{ queryId, chosenIds[] }` → 融合答案
- `GET /trips/cultivation/wisdom/history`

## 5. Prompt 工程
详见 F37.03-PRD-AI融通问答Prompt工程.md

## 6. 限流与安全
- Rate Limit: 10 次/小时/用户
- 敏感词过滤 (政治/暴力/低俗) — 复用 moderation 模块
- LLM 超时 15s, 超时降级为 3 大文化 (禅/基/道)
- 所有答案缓存 Redis 7天 (相同问题命中)

## 7. 验收
- [ ] 12 路并行 < 15s
- [ ] SSE 流式稳定
- [ ] 融合答案质量 A/B 测试 ≥ 4.0/5
- [ ] 敏感词拦截
- [ ] 历史会话翻页
- [ ] 一键保存到因缘日志
