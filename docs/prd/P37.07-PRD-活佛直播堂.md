## 4. 技术
- 真人直播: RTMP 推流 + HLS 分发 (可选用阿里云/腾讯云)
- AI 分身: 预生成 MP4 + Live2D/数字人头像
- 字幕: WebVTT 实时同步
- 弹幕: WebSocket 复用 M30 chat 模块
- 打赏: 复用 payment 模块 + membership 积分
