# P37.03-PRD-每日一印修行页

> 圆满之路 A3 轴 | 路由 /trips/cultivation/daily-seal | 30印 × 12文化 = 360日课

## 1. 页面定位
每日 2 次 (晨 06:00 / 暮 18:00) 推送一印, 含引导音频 + 书法印面 + 短文解说 + 反思输入框. 打卡形成连击.

## 2. 内容结构
- **印**: 复用 M03 曹溪三十印 (五系: 初印→中印→印果→成道→归源)
- **文化映射**: 每印 × 12文化, 共 360 条内容
- **推荐算法**: 境界 + 主修 + 历史打卡 → 今日印
- **冷却**: 同一印 30 天内不重复

## 4. 交互
- 音频使用 HTML5 Audio, 听完 ≥ 80% 才能打卡
- 打卡后曼陀罗连击光晕 +1
- 7连击 +50积分, 21连击 +200积分, 100连击解锁"百日印成"成就
- 跳过可选但不记连击

## 5. API
- `GET /trips/cultivation/daily-seal/today?session=MORNING|EVENING`
- `POST /trips/cultivation/daily-seal/practice` `{ sealId, audioSec, reflection }`
- `GET /trips/cultivation/daily-seal/streak`

## 6. 内容种子要求
- 30印 × 12文化 = 360 条
- 每条: 标题 + 解说 (300字) + 音频URL (12min标准) + 书法印面SVG
- Wave 5 批量生成, 允许 MVP 先上禅宗+道教+基督 90条, 其余渐进
