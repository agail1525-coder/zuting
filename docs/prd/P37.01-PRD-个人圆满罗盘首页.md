# P37.01-PRD-修行罗盘首页

> M37 修行系统 A1 轴 | 路由 `/trips/cultivation` | 需 CultivationAccessGuard | 禅宗主视角 × 12文化同构

## 1. 页面定位
**修行子系统的主控台**, 位于 `/trips/cultivation` (进入 /trips 页后点第4卡「我的修行」跳转). 不取代主站首页, 仅对已授权修行者开放. 未授权访问 → redirect `/trips/cultivation/apply`.

核心体验: 一张动态曼陀罗 + 今日一步指引 + 当前境界卡 + 融修文化切换.

## 2. 用户故事
- 作为初觉境界的禅宗学员, 我打开罗盘看到当前是"寻牛", 今日指引是"禅坐10分钟 + 读《无门关》第一则", 我能一键进入修行.
- 作为修佛三年想融修基督文化的用户, 我能在罗盘加入"基督文化"作为融修, 当日指引变成"禅坐 + 读《登山宝训》并对比".

## 4. 交互细节

- 曼陀罗使用 D3 + Framer Motion, 连击天数越高光晕越亮
- 晋阶时触发金色粒子爆裂动画 + 钟声
- 融修切换不覆盖主修, 并列显示
- 未完成今日一步时, 中心境界字呈灰度; 全部完成后金色

## 5. API
- `GET /trips/cultivation` 返回 `{ realm, oxStage, todaySteps[], mandalaState, streakDays }`
- `POST /trips/cultivation/step/:id/complete` 完成一步

## 6. 三态
- Loading: 曼陀罗骨架 + 墨点加载
- Empty: 未初始化 → CTA "开启我的圆满之路" → 跳 M36 评估
- Error: 降级纯文字今日指引
