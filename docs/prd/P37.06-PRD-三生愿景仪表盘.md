# P37.06-PRD-三生愿景仪表盘

> 圆满之路 A6 轴 | 路由 /trips/cultivation/three-lives | 个人圆满 + 家庭幸福 + 企业兴旺 三轴合一

## 1. 页面定位
将 M34 (个人圆满) + M35 (家庭幸福) + M32 (团队文化) 三个独立模块 **在一张仪表盘上统一呈现**, 让用户看到三生如何相互成就.

## 2. 核心理念
```
个人圆满 ←→ 家庭幸福 ←→ 企业兴旺
    ↑                         ↓
    ╰─────── 愿财双圆 ────────╯
```
三者任一失衡都无法真正圆满. 仪表盘的价值是 **强迫用户同时关注三轴**.

## 3. 布局
```
┌─────────────────────────────────────┐
│           三生愿景仪表盘             │
├─────────────────────────────────────┤
│                                       │
│   [个人]        [家庭]       [企业] │

```

## 4. AI 对齐引擎
输入: 三轴当前进度 + 近30天因缘日志 + 信仰力评分
输出: 下月重点轴建议 + 具体行动3条

## 5. API
- `GET /trips/cultivation/three-lives`
- `PUT /trips/cultivation/three-lives` 保存三轴目标文字
- `POST /trips/cultivation/three-lives/align` AI 对齐建议
- `GET /trips/cultivation/three-lives/export-pdf`

## 6. 数据关联
- 个人: 读 M34 PersonalGrowthEnrollment
- 家庭: 读 M35 FamilyHarmonyEnrollment
- 企业: 读 M32 TeamCultureEnrollment
