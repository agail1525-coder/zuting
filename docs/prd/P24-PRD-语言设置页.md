# P24-PRD-语言设置页

> 页面: 语言设置/切换
> 对标: Booking.com语言选择器 / Trip.com语言面板
> Sprint: D1 | 优先级: P0

---

## 1. 概述

提供用户友好的语言切换界面，覆盖Web Header下拉、Mobile设置页、小程序设置页。

---

## 2. Web — Header语言切换器

### 2.1 交互
- Header右上角显示当前语言图标+代码 (🌐 中文)
- 点击弹出下拉面板
- 7语言列表，当前语言高亮
- 选择后即时切换，存localStorage

### 2.2 语言列表
| 图标 | 语言 | 代码 |
|------|------|------|
| 🇨🇳 | 简体中文 | zh-CN |
| 🇺🇸 | English | en |
| 🇯🇵 | 日本語 | ja |
| 🇰🇷 | 한국어 | ko |
| 🇹🇭 | ไทย | th |
| 🇮🇳 | हिन्दी | hi |
| 🇸🇦 | العربية | ar |

---

## 3. Mobile — 设置页语言选项

- Profile → 设置 → 语言
- 列表选择，当前语言勾选
- 切换后App内文本即时更新

---

## 4. Miniprogram — 设置页语言选项

- 我的 → 设置 → 语言
- Picker选择器
- 切换后页面文本更新

---
