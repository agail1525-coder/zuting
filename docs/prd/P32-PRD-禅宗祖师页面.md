## 1. 概述

创建专属禅宗祖师频道，以曹洞宗传承法脉为核心，展示从洞山良价(第一世)到当代(第五十二世)的完整传承谱系。每位祖师配备AI生成水墨风头像，详情页包含公案、成就、祖庭、师徒关系、经典语句、传世著作、禅宗流派等丰富内容。

### 1.1 核心价值

| 维度 | 描述 |
|------|------|
| 教育价值 | 完整呈现曹洞宗1200年传承史 |
| 文化价值 | 保存和传播禅宗文化遗产 |
| 旅行引导 | 每位祖师关联祖庭寺院，引导朝圣 |
| 差异化 | 全网最完整的曹洞宗传承可视化 |

## 2. 数据模型扩展

### 2.1 Patriarch模型新增字段

```prisma
model Patriarch {
  // ...existing fields...
  
  // NEW: 禅宗特有字段
  school        String?    // 禅宗流派: 曹洞宗/临济宗/云门宗/法眼宗/沩仰宗
  generation    Int?       // 传承世次 (曹洞宗内)
  teacherId     String?    // 师承 (self-reference)
  koans         Json?      // 公案 [{title, content, source}]
  achievements  String?    @db.Text  // 主要成就
  templeNames   Json?      // 关联祖庭 [{name, nameEn, role, location}]
  classicQuotes Json?      // 经典语句 [string]
  works         Json?      // 传世著作 [{title, description}]
}
```

## 6. AI生成头像规范

- 风格: 中国传统水墨画 (写意风)
- 背景: 山水/禅院/竹林/松树
- 服饰: 汉传佛教僧袍
- 色调: 黑白水墨为主，淡赭石点缀
- 尺寸: 512×512px, WebP格式
- 命名: `/images/zen-patriarchs/{slug}.webp`
