# F37.02-PRD-十牛图×12文化符号映射表 + 深修包注册表

> M37 跨宗合愿核心字典 | 84 条境界映射 + 12 个深修包 | 禅宗为主, 12 文化平等并列

## 1. 目的与原则

### 1.1 目的
为 A1 罗盘 / A2 十牛图路径 / A4 融通问答 / 算法引擎 提供跨文化符号翻译表. 用户切换主修/融修文化时, 同一境界显示对应文化的原生符号和经典表达.

### 1.2 设计原则 (与 M37 §3.5.4 一致)
1. 不改信仰 — 帮助用户在自己传统中走得更深, 不改变信仰归属
2. 平等显示 — 12 文化平等
3. 禅宗主轴 — 默认 UX 锚点
4. 不混搭真理 — 各家原汁原味
5. 不评判优劣
6. 学者级严谨 — 必有原典出处

## 2. 数据结构

```prisma
model OxCultureMapping {
  id          String   @id @default(cuid())
  realmId     Int      // 1-7 七境界
  oxStage     Int      // 1-10 十牛图阶段 (压缩到七境界时多对一)
  tradition   CultureCategory  // 12文化枚举
  symbolName  String              // 符号名 (中文)
  symbolNameNative String?        // 原文 (Tat Tvam Asi / Tawhid 等)
  originalText String   @db.Text  // 原典引文
  source      String              // 经典出处
  explanation String   @db.Text   // 100-200 字解说
  imageUrl    String?
  hasDeepModule Boolean @default(false)  // 是否有该境界的深修包
  deepModuleCode String?           // DM-01..DM-12
  reviewStatus String   @default("DRAFT") // DRAFT|REVIEW|APPROVED
  reviewedBy  String?              // 文化顾问 userId
  @@unique([realmId, tradition])
  @@index([tradition, realmId])
}

model CultivationDeepModule {
  id          String   @id @default(cuid())
  code        String   @unique  // DM-01..DM-12
  name        String              // 苏菲旋舞 Sema
  tradition   CultureCategory
  applicableRealms Int[]          // [5, 6] 适用境界
  durationDays Int                // 7-49
  description String   @db.Text
  contentJson Json                // { days: [{ title, audio, text, meditation, checkin }] }
  unlockBy    String              // AUTO_ON_BLEND | MANUAL | PURCHASE
  isActive    Boolean  @default(true)
}

model UserDeepModuleProgress {
  id          String   @id @default(cuid())
  userId      String
  moduleCode  String
  currentDay  Int      @default(1)
  status      String   @default("IN_PROGRESS") // IN_PROGRESS|COMPLETED|PAUSED
  startedAt   DateTime @default(now())
  completedAt DateTime?
  @@unique([userId, moduleCode])
}
```

## 5. 内容生产策略

### 5.1 优先级
- **MVP (Wave 1)**: 禅宗+道教+基督 三列共 21 条 + DM-03 + DM-10
- **Wave 2**: 儒家+藏传+伊斯兰 共 21 条 + DM-01 + DM-04
- **Wave 3**: 印度教+犹太 共 14 条 + DM-02 + DM-05 + DM-06 + DM-08
- **Wave 4**: 锡克+巴哈伊+神道+原住民 共 28 条 + DM-07 + DM-09 + DM-11 + DM-12

### 5.2 内容审核流
```
内容草稿 (AI辅助)
  → 文化顾问初审 (该文化的学者/导师)
  → 跨文化平等性检查 (Admin 团队)
  → reviewStatus = APPROVED
  → 入库供 OxCultureMapping 使用
```

### 5.3 文化顾问团 (推荐配置)
- ZEN: 禅宗法师 1
- TAOISM: 道协老师 1
- CHRISTIANITY: 神学院教授 1
- ISLAM: 苏菲学者 1
- HINDUISM: Vedanta 学者 1
- JUDAISM: 拉比 + 卡巴拉学者 各 1
- 其他 6 文化各 1 名学术顾问

## 6. 使用方式

### 6.1 罗盘页 (P37.01)
```typescript
const mapping = await prisma.oxCultureMapping.findUnique({
  where: { realmId_tradition: { realmId: journey.currentRealm, tradition: journey.primaryTradition } }
});
// → 显示中心符号 + 原典引文
```

### 6.2 十牛图详情抽屉 (P37.02)
显示 12 列并排, 每列一文化, 突出当前主修.

### 6.3 融通问答 (P37.04 + F37.03)
将映射表注入 Prompt 作为各祖师的人设锚点, 确保回答时引用对应原典.

### 6.4 深修包激活
```
用户在罗盘 → 切换主修/融修 → 检查该文化是否有未完成 DM
  → 提示 "解锁新深修包: 苏菲旋舞 14 日"
  → 用户接受 → 创建 UserDeepModuleProgress
```
