## 3. 布局
横向滚动长卷 (PC) / 竖向长屏 (Mobile). 水墨画 + 书法标题 + 当前位置红点.

点击任一图 → 弹出详情抽屉, 含 12 文化对应符号 + 跳转按钮.

## 4. 晋阶规则
- 只能向前一阶, 不可跳跃
- 满足里程碑 + 当前阶段日课累计 ≥ 21 天 → 解锁晋阶
- 晋阶触发 POST /trips/cultivation/ox-path/advance, 动画: 云气翻卷 + 钟磬声

## 5. API
- `GET /trips/cultivation/ox-path` → `{ currentStage, stages[10], unlockedAt[] }`
- `POST /trips/cultivation/ox-path/advance` → 校验条件, 写审计日志
