## 一、竞品对标分析

| 竞品 | 详情页亮点 | 借鉴点 |
|------|------------|-------|
| **Expedia Package Detail** | 全宽Hero图，价格横幅固定底部，按日行程可展开折叠，包含内容图标化清单 | 固定底部预订栏、行程可折叠、图标清单 |
| **Trip.com跟团详情** | 出发日期日历+剩余名额倒计时，人数-/+选择器，详细费用包含/不含说明 | 日历选座、名额紧张感、费用明细 |
| **Airbnb体验详情** | 沉浸式照片画廊（可浏览多张），主持人介绍卡片，评价精选，类似体验推荐 | 多图画廊、主持人/导游展示、评价区块 |
| **Booking.com度假套餐** | 价格构成明细（每晚/每日价格拆解），灵活取消政策，信任度徽章 | 价格拆解透明、取消政策显著、信任标识 |
| **携程自由行详情** | 醒目的"立即预订"悬浮按钮，游客评价图文混排，问题FAQ展开 | 悬浮CTA、评价图文、FAQ |

## 三、功能清单

### 3.1 P0 核心区块

| 区块ID | 区块名称 | 说明 |
|--------|---------|------|
| B-P19-01 | Hero封面图 | 全宽大图，支持点击查看大图 |
| B-P19-02 | 基本信息 | 套餐名称、类型标签、时长、最大人数 |
| B-P19-03 | 价格展示 | 原价、会员价、每人价格说明 |
| B-P19-04 | 包含内容 | 4项内容图标+文字说明，明确展示 |
| B-P19-05 | 按日行程 | 时间轴样式，每天标题+活动列表，可展开 |
| B-P19-06 | 预订表单 | 人数选择、日期选择、联系人信息 |
| B-P19-07 | 固定底部预订栏 | 总价 + 立即预订按钮，滚动时固定 |

### 3.2 P1 重要区块

| 区块ID | 区块名称 | 说明 |
|--------|---------|------|
| B-P19-08 | 相关圣地/祖庭 | 套餐关联entityIds的目的地卡片 |
| B-P19-09 | 会员专属价模块 | 等级≥3时展示节省金额，引导未达级用户升级 |
| B-P19-10 | 预订须知 | 取消政策、出发要求、注意事项 |
| B-P19-11 | 分享按钮 | 复制链接/分享到微信/生成海报 |

### 3.3 P2 扩展区块

| 区块ID | 区块名称 | 说明 |
|--------|---------|------|
| B-P19-12 | 用户评价 | 完成订单用户的评分+文字+图片 |
| B-P19-13 | 导游介绍 | 若套餐含导游，展示导游简介和评分 |
| B-P19-14 | 相似推荐 | 同类型其他套餐横向滚动卡片 |
| B-P19-15 | 问题FAQ | 常见问题折叠面板 |

## 七、API调用

| 操作 | API端点 | 时机 |
|------|---------|------|
| 获取套餐详情 | GET /api/packages/:id | 页面加载 |
| 获取用户会员信息 | GET /api/membership/me | 页面加载（已登录） |
| 提交预订 | POST /api/packages/:id/book | 确认预订 |

## 八、验收标准

- [ ] Hero封面图全宽展示，加载中显示骨架屏
- [ ] 套餐基本信息（名称/类型/时长/人数）完整展示
- [ ] includes 4项图标：有的展示原色，无的展示灰色划线
- [ ] 行程按天展示，每天可展开/折叠查看详细活动
- [ ] 预订表单：日期不能选今天之前，人数1-50范围
- [ ] 固定底部预订栏在页面滚动时始终可见（Web sticky/Mobile固定）
- [ ] 人数变化时底部总价实时更新
- [ ] Lv3+用户看到会员价并显示节省金额
- [ ] 预订成功后跳转订单详情页（或成功页）
- [ ] 相关圣地卡片点击跳转对应圣地详情页
- [ ] 404处理：套餐不存在或已下架时显示友好错误页

## 九、技术约束

```
[TC-P19-01] Web端使用 generateStaticParams 预渲染热门套餐，配合 revalidate = 3600
[TC-P19-02] 预订表单使用 react-hook-form + zod 校验
[TC-P19-03] 日期选择器: Web用 react-day-picker，Mobile用 DateTimePicker
[TC-P19-04] 固定底部: Web用 sticky bottom bar + shadow，Mobile用 SafeAreaView包裹
[TC-P19-05] 行程展开动画: Framer Motion (Web) / LayoutAnimation (Mobile)
[TC-P19-06] 会员价判断在前端做，同时后端在book接口校验会员等级并使用正确价格
[TC-P19-07] 提交预订前必须登录，未登录跳转登录后返回本页
```
