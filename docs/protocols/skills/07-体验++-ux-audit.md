## §3 领域规则 — 8维度UX扫描

### T1 三态完整性 (Loading/Error/Empty)
```
[UX-01] 每个数据驱动的页面必须有loading态 (骨架屏/Spinner)
[UX-02] 每个API调用必须有error态 (错误提示+重试按钮)
[UX-03] 每个列表页必须有empty态 (友好插图+引导文案)
[UX-04] 三态之间的转换必须平滑 (无闪烁/跳动)
[UX-05] loading态不能无限持续 → 超时后显示错误态
```

### T2 响应式布局
```
[UX-06] 移动端(≤768px)布局不破裂，文本不溢出
[UX-07] 平板(768-1024px)合理利用空间
[UX-08] 桌面端(>1024px)内容区域有最大宽度约束
[UX-09] 图片在不同尺寸下保持比例，不变形
[UX-10] 表格在移动端有横向滚动或卡片替代方案
```

### T3 国际化(i18n)
```
[UX-11] 所有用户可见文本使用i18n键 → 禁止硬编码中文
[UX-12] 7语言JSON文件key集合必须完全一致
[UX-13] 阿拉伯语(ar)页面支持RTL布局
[UX-14] 日期/时间显示根据语言locale格式化
[UX-15] 货币符号跟随用户设置，不硬编码¥
[UX-16] 长翻译文本(如德语/阿拉伯语)不破坏布局
```

### T4 交互反馈
```
[UX-17] 按钮点击后有视觉反馈(ripple/颜色变化/loading)
[UX-18] 表单提交后有成功/失败toast
[UX-19] 删除操作前有确认对话框
[UX-20] 列表操作(收藏/评价)后有即时视觉反馈
[UX-21] 页面切换有平滑过渡 (非突然闪白)
```

### T5 可访问性(A11y)
```
[UX-22] 图片有描述性alt文本
[UX-23] 按钮有aria-label (特别是图标按钮)
[UX-24] 表单字段有关联的label
[UX-25] 颜色对比度满足WCAG AA标准 (暗色主题尤其注意)
[UX-26] 键盘可导航 (Tab/Enter/Escape)
```

### T6 导航与信息架构
```
[UX-27] 用户始终知道自己在哪 (面包屑/Tab高亮/标题)
[UX-28] 返回操作始终可用且指向合理的上级
[UX-29] 搜索/筛选结果清晰标注数量和条件
[UX-30] 分页信息清晰 (当前页/总页/每页数量)
```

### T7 感知友好度
```
[UX-31] 错误信息用友好语言而非技术术语
[UX-32] 空状态有引导性文案 (如"开始你的朝圣之旅")
[UX-33] 宗教内容展示尊重各信仰文化 (无冒犯性设计)
[UX-34] 价格信息清晰透明 (含税/不含税/货币单位)
```

### T8 性能感知
```
[UX-35] 首屏内容在2秒内可见 (骨架屏→真实内容)
[UX-36] 图片懒加载 (非首屏图片)
[UX-37] 列表虚拟滚动 (>50条数据)
[UX-38] 搜索有防抖 (debounce ≥300ms)
[UX-39] 页面切换无白屏 (SSR/预渲染)
```

## §4 执行流程

```
Step 0: 确定目标页面 + 读取设计规范
Step 1: T1-T2 基础扫描 (三态+响应式)
  → 检查每个页面的loading/error/empty态
  → 检查关键断点(768px/1024px)布局
Step 2: T3-T4 交互扫描 (i18n+反馈)
  → 对比7语言JSON key集合
  → 检查操作反馈(按钮/表单/删除)
Step 3: T5-T8 深层扫描 (A11y+导航+感知+性能)
  → 检查alt/aria-label/label
  → 检查导航结构/面包屑
  → 检查性能感知指标
Step 4: 修复 (Executor)
  → P0: 三态缺失/布局破裂
  → P1: i18n缺失/无反馈
  → P2: A11y/性能优化
Step 5: 验证 (Verifier)
  → 重跑检测
  → tsc + 构建验证
```

## §6 工具治理

```
允许工具:
  - Read: 读取组件代码/i18n文件
  - Grep: 搜索i18n key/alt/aria-label
  - Glob: 发现页面和组件文件
  - Edit: 修复体验问题
  - Bash: 构建验证

核心检测命令:
  i18n完整性:   对比 zh-CN.json 与 en.json 的key差集
  硬编码文本:   Grep ">[^<]*[\u4e00-\u9fff]" --glob "*.tsx" (中文硬编码)
  alt缺失:      Grep "<img(?!.*alt)" --glob "*.tsx"
  loading态:    Grep "isLoading|loading|Spinner|Skeleton" --glob "*.tsx"
  empty态:      Grep "empty|暂无|no data|No.*found" --glob "*.tsx"
```
