## 1. Buddhism / 佛教

### 1.1 Respectful Representation / 尊重表述

**Proper Terminology / 正确术语:**
| Correct / 正确 | Incorrect / 不正确 | Notes / 备注 |
|---|---|---|
| Temple (寺/庙) | Buddhist church | Use "temple" or "monastery" depending on function / 根据功能使用"寺院"或"禅院" |
| Monastery (寺院) | Buddhist convent | For monastic communities / 用于僧侣社区 |
| Vihara (精舍) | Shrine house | Pali term for monastic dwelling / 巴利语指僧人居所 |
| Stupa (塔/佛塔) | Pagoda (acceptable in East Asia) | "Pagoda" is acceptable for East Asian tower-style structures / "塔"可用于东亚塔形建筑 |
| Dharma (法) | Buddhist law | Never translate as "law" / 不可译为"法律" |
| Sangha (僧伽) | Buddhist community | Refers specifically to the monastic order / 特指僧团 |
| Sutra (经) | Scripture | "Sutra" is the preferred term / "经"是首选术语 |
| Buddha image/statue (佛像) | Idol | NEVER use "idol" — deeply offensive / 绝不使用"偶像"——极具冒犯性 |

**Sacred Text Handling / 经典处理:**
- Buddhist sutras should never be placed on the floor or stepped over. / 佛经不可放在地上或跨越。
- When displaying sutra text, use respectful formatting — no truncation mid-verse. / 显示经文时使用恭敬格式——不可在偈颂中间截断。
- The Pali Canon, Chinese Tripitaka, and Tibetan Kangyur each have distinct transliteration norms. / 巴利三藏、汉文大藏经和藏文甘珠尔各有不同的转写规范。

**Image Guidelines / 图像指南:**
- Buddha images are sacred. Never use them in humorous, commercial, or disrespectful contexts. / 佛像是神圣的，不可用于幽默、商业或不敬的场合。
- Some countries (e.g., Thailand, Myanmar) have laws against inappropriate use of Buddha images. / 部分国家（如泰国、缅甸）有法律禁止不当使用佛像。
- Never show Buddha images below waist level in layouts, or cropped at the neck. / 布局中不可将佛像放在腰部以下，或从颈部裁剪。
- Tattoos of Buddha are controversial in many Buddhist countries. Do not promote or showcase them. / 佛陀纹身在许多佛教国家存在争议，不要推广或展示。

**Color Symbolism / 色彩象征:**
- Saffron/gold (藏红/金色): Renunciation, wisdom — primary monastic robe color in Theravada / 出离、智慧——上座部僧侣袈裟主色
- White (白色): Purity, mourning (in some traditions) / 纯净、哀悼（某些传统中）
- Maroon/dark red (栗色/暗红): Used in Mahayana and Vajrayana traditions / 大乘和金刚乘传统使用
- Blue (蓝色): Healing Buddha (Medicine Buddha) / 药师佛
- Platform color: `#F59E0B` (amber/saffron) / 平台色: `#F59E0B`（琥珀/藏红色）

### 1.2 Pilgrimage Etiquette / 朝圣礼仪

**Dress Code / 着装规范:**
- Cover shoulders and knees in all temples. / 在所有寺院中须遮盖肩膀和膝盖。
- White clothing is often appropriate for lay practitioners attending ceremonies. / 白色衣物通常适合在家众参加仪式。
- Remove hats inside temple halls. / 在殿堂内须脱帽。
- Avoid revealing or tight-fitting clothing. / 避免暴露或紧身的衣物。

**Photography Rules / 拍摄规定:**
- Always ask permission before photographing inside temple halls. / 在殿堂内拍照前务必征得许可。
- NEVER take selfies with or turn your back to a Buddha statue. / 绝不与佛像自拍或背对佛像。
- Flash photography is usually prohibited inside shrine rooms. / 殿堂内通常禁止使用闪光灯。
- Many Chinese and Japanese temples allow photography; Thai and Burmese temples vary. / 许多中日寺院允许拍照；泰缅寺院规定不一。

**Prayer Times / 课诵时间:**
- Morning chanting: typically 4:00-6:00 AM / 早课: 通常凌晨4:00-6:00
- Evening chanting: typically 4:00-6:00 PM / 晚课: 通常下午4:00-6:00
- Visitors should remain silent during chanting periods. / 课诵期间访客应保持安静。

**Offering Customs / 供养习俗:**
- Common offerings: flowers, incense, candles, fruit, clean water. / 常见供品：花、香、烛、水果、净水。
- In Theravada: food offerings to monks must be given before noon. / 上座部：供僧食物须在中午前供养。
- Never offer meat or alcohol at Buddhist sites. / 不可在佛教场所供养荤腥或酒精。

**Gender-Specific Rules / 性别相关规定:**
- Women must NOT touch monks in Theravada traditions. / 在上座部传统中，女性不可触碰僧侣。
- Women may not be allowed in certain inner sanctums (e.g., some Thai ordination halls). / 女性可能不被允许进入某些内殿（如泰国部分戒堂）。
- When passing objects to monks, women should place them on a cloth or surface, not hand directly. / 女性递物给僧侣时应放置在布或台面上，不可直接递交。

**Shoe Removal / 脱鞋规范:**
- Shoes MUST be removed before entering any Buddhist temple hall. / 进入任何佛教殿堂前必须脱鞋。
- Some complexes require shoe removal at the outer gate. / 某些寺院群在外门即须脱鞋。
- Carry a bag for shoes; do not leave them blocking pathways. / 随身携带鞋袋；不可将鞋置于通道中。

**Dietary Considerations / 饮食注意:**
- Many Buddhist sites are vegetarian zones. / 许多佛教场所为素食区域。
- Mahayana tradition: strictly vegetarian. / 大乘传统：严格素食。
- Theravada: monks eat what is offered, including meat, but killing is prohibited. / 上座部：僧侣食用供养食物（含肉），但禁止杀生。
- No alcohol at or near temple grounds. / 寺院范围内及附近禁止饮酒。

### 1.3 Calendar & Holy Days / 历法与节日

| Holiday / 节日 | Approximate Date / 大致日期 | Impact / 影响 |
|---|---|---|
| Vesak (卫塞节) | May full moon / 五月满月 | Busiest day; sites crowded but festive. Special access to ceremonies. / 最繁忙日；人多但氛围庄严，可参加特殊仪式 |
| Asalha Puja (阿萨拉布差) | July full moon / 七月满月 | Start of Buddhist Lent; monks begin retreat. / 佛教结夏安居开始 |
| Kathina (迦提那) | October-November / 十月至十一月 | End of Lent; robe-offering ceremonies. / 安居结束；供袈裟仪式 |
| Chinese Buddhist holidays (中国佛教节日) | Lunar calendar / 农历 | Guanyin's Birthday (2/19, 6/19, 9/19 lunar), Buddha's Birthday (4/8 lunar) / 观音诞辰、佛诞 |
| Obon (盂兰盆节) | August 13-15 / 八月13-15日 | Japanese ancestor festival; temples very active. / 日本祭祖节；寺院非常活跃 |
| Uposatha (布萨日) | Every full/new moon / 每个满月/新月 | Enhanced practice days; some sites have restricted access. / 加行日；部分场所限制进入 |

**Best times to visit / 最佳参访时间:** Early morning (before 8 AM) for quiet contemplation; festival days for cultural immersion. / 清晨（8点前）适合静修；节日适合文化体验。

**Worst times / 不宜时间:** Avoid during intensive retreat periods (Vassa/Rain Retreat, July-October) if seeking monk interaction. / 如欲与僧侣互动，避开密集闭关期。

### 1.4 Language Considerations / 语言注意事项

**Sacred Languages / 神圣语言:** Pali (巴利语), Sanskrit (梵语), Classical Chinese (文言文), Japanese (日语), Korean (韩语), Thai (泰语)

**Transliteration Standards / 转写标准:**
- Pali: use the Pali Text Society romanization (e.g., "Dhamma" not "Dharma" for Theravada). / 巴利语：使用巴利圣典协会罗马化
- Sanskrit: use IAST (International Alphabet of Sanskrit Transliteration). / 梵语：使用IAST
- Chinese: use Pinyin with tone marks for pronunciation guides. / 中文：使用带声调的拼音

**Terms NOT to translate / 不翻译的术语:** Dharma (法), Karma (业), Nirvana (涅槃), Bodhi (菩提), Sangha (僧伽), Sutra (经), Bodhisattva (菩萨), Mandala (曼陀罗), Zen (禅)

### 1.5 Sensitivity Red Flags / 敏感问题

- NEVER compare Buddhism to "philosophy" dismissively — it is a complete spiritual path. / 不可将佛教贬低为"哲学"——它是完整的修行道路。
- Avoid implying Buddhism is "atheistic" — this is a Western oversimplification. / 避免暗示佛教是"无神论"——这是西方的过度简化。
- Be careful with the term "Hinayana" (小乘) — it is considered derogatory by Theravada practitioners. Use "Theravada" (上座部). / 慎用"小乘"一词——上座部修行者视为贬义，应使用"上座部"。
- The Dalai Lama is a political as well as religious figure; handle content about him with awareness of geopolitical sensitivities. / 达赖喇嘛同时具有政治和宗教身份，须注意地缘政治敏感性。
- Rohingya crisis involves Buddhist-Muslim tensions in Myanmar — handle with extreme care. / 罗兴亚危机涉及缅甸佛教-穆斯林紧张关系——须极度谨慎处理。
- Commercialization of Buddhism (Buddha bars, Buddha-head decor) is offensive to practitioners. / 佛教商业化（佛陀酒吧、佛头装饰）对修行者具有冒犯性。

**Content Moderation Keywords / 内容审核关键词:** idol worship, fake religion, superstition, Buddha bar, Buddha tattoo, fat Buddha, laughing Buddha (as commercial), cult

## 2. Taoism / 道教

### 2.1 Respectful Representation / 尊重表述

**Proper Terminology / 正确术语:**
| Correct / 正确 | Incorrect / 不正确 | Notes / 备注 |
|---|---|---|
| Taoist temple/Guan (道观/宫) | Taoist church | "Guan" (观) is the traditional term / "观"是传统称谓 |
| Dao (道) | The Way (acceptable as translation) | Original term preferred in spiritual context / 灵性语境中首选原词 |
| Daoist priest (道士/道长) | Taoist monk | "Priest" or "Master" more accurate; "monk" implies Buddhist usage / "道士"或"道长"更准确 |
| De (德) | Virtue/Power | Both translations are valid; use in context / 两种翻译均可，视语境使用 |
| Wu Wei (无为) | Doing nothing | Means "effortless action," NOT "inaction" / 意为"自然而然"，非"无所作为" |
| Qi (气) | Chi / Energy | "Qi" is preferred modern romanization / "Qi"是首选现代罗马化 |

**Sacred Text Handling / 经典处理:**
- The Dao De Jing (道德经) has thousands of translations — always cite the source translation. / 《道德经》有数千种翻译——须注明翻译来源。
- The Zhuangzi (庄子) should be identified as a philosophical text, not just "stories." / 《庄子》应被定性为哲学文本，而非仅仅是"故事"。
- Taoist liturgical texts (科仪) are considered sacred and should not be excerpted casually. / 道教科仪文本被视为神圣，不可随意摘录。

**Image Guidelines / 图像指南:**
- Taoist deity images should be treated with the same respect as Buddhist ones. / 道教神像应受到与佛像同等的尊重。
- The Yin-Yang symbol (太极图) is sacred — do not use it as mere decoration. / 太极图是神圣的——不可仅作装饰使用。
- The Bagua (八卦) arrangement matters — do not randomly rearrange trigrams. / 八卦排列有讲究——不可随意重排。

**Color Symbolism / 色彩象征:**
- Black and white (黑白): Yin-Yang duality / 阴阳对立
- Green/teal (青/蓝绿): Nature, vitality, the Wood element / 自然、生命力、木行
- Red (红): Auspicious, used in Taoist ritual / 吉祥，用于道教仪式
- Yellow (黄): Earth element, the center / 土行、中央
- Platform color: `#10B981` (emerald/jade) / 平台色: `#10B981`（翡翠/玉色）

### 2.2 Pilgrimage Etiquette / 朝圣礼仪

**Dress Code / 着装规范:**
- Modest clothing; avoid black at celebratory events (associated with mourning in some contexts). / 着装端庄；庆典活动避免黑色（某些场合与丧事相关）。
- Some temples require no shorts above the knee. / 部分道观要求不可穿膝上短裤。

**Photography Rules / 拍摄规定:**
- Generally more relaxed than Buddhist temples but ask first in inner sanctums. / 通常比佛教寺院宽松，但在内殿须先询问。
- Do not photograph Taoist rituals without permission. / 未经许可不可拍摄道教仪式。

**Offering Customs / 供养习俗:**
- Common: incense (three sticks), fruit, tea, rice wine. / 常见：香（三炷）、水果、茶、米酒。
- Paper offerings (纸钱/元宝) are burned at designated areas only. / 纸钱/元宝仅在指定区域焚烧。
- Do not blow out incense — wave it or let it extinguish naturally. / 不可吹灭香——挥动或自然熄灭。

**Shoe Removal / 脱鞋规范:**
- Required in main halls of most Taoist temples. / 大多数道观的主殿须脱鞋。
- Step over the threshold (门槛), do not step on it. / 须跨过门槛，不可踩踏。

**Dietary Considerations / 饮食注意:**
- Quanzhen (全真) Taoists are strictly vegetarian and teetotal. / 全真道士严格素食且不饮酒。
- Zhengyi (正一) Taoists may consume meat except during fasting periods. / 正一道士可食肉，但斋戒期间除外。
- Major sacred mountains (Wudang, Qingcheng) have vegetarian restaurants. / 主要道教名山（武当、青城）有素食餐厅。

### 2.3 Calendar & Holy Days / 历法与节日

| Holiday / 节日 | Approximate Date / 大致日期 | Impact / 影响 |
|---|---|---|
| Chinese New Year (春节) | Jan-Feb (lunar) / 正月初一 | Temples very crowded; special rituals. / 道观非常拥挤；特殊法事 |
| Lantern Festival (元宵节) | 15th of 1st lunar month / 正月十五 | Temple fairs and lantern displays. / 庙会和灯展 |
| Laozi's Birthday (老君诞) | 2/15 lunar / 农历二月十五 | Major Taoist celebration. / 道教重大庆典 |
| Zhongyuan Festival (中元节) | 7/15 lunar / 农历七月十五 | Ghost Festival; special rituals for ancestors. / 鬼节；祖先特殊法事 |
| Double Ninth (重阳节) | 9/9 lunar / 农历九月初九 | Mountain climbing tradition; temples on mountains are busy. / 登高传统；山上道观繁忙 |

### 2.4 Language Considerations / 语言注意事项

**Sacred Language:** Classical Chinese (文言文)

**Transliteration:** Use Pinyin with tone marks. "Dao" not "Tao" (modern standard), though "Tao" is acceptable in established terms like "Tao Te Ching."

**Terms NOT to translate / 不翻译的术语:** Dao (道), De (德), Qi (气), Wu Wei (无为), Yin (阴), Yang (阳), Taiji (太极), Bagua (八卦), Xian (仙), Neidan (内丹), Wudang (武当)

### 2.5 Sensitivity Red Flags / 敏感问题

- Do NOT conflate Taoism with "Chinese folk religion" — they overlap but are distinct. / 不可将道教与"中国民间信仰"混同——二者有交集但不同。
- Wu Wei does NOT mean "laziness" or "passivity." / 无为不等于"懒惰"或"消极"。
- Feng Shui (风水) and martial arts are related to but not the entirety of Taoism. / 风水和武术与道教相关但非道教全部。
- Avoid "mysterious Oriental wisdom" stereotypes. / 避免"神秘东方智慧"刻板印象。
- The relationship between Taoism and the Chinese government has complex historical dimensions. / 道教与中国政府关系有复杂的历史维度。

**Content Moderation Keywords:** cult, black magic, sorcery, fortune telling scam, fake medicine, quack

## 3. Christianity / 基督教

### 3.1 Respectful Representation / 尊重表述

**Proper Terminology / 正确术语:**
| Correct / 正确 | Incorrect / 不正确 | Notes / 备注 |
|---|---|---|
| Church (教堂) | Christian temple | "Church" for the building / "教堂"指建筑 |
| Cathedral (大教堂) | Big church | A cathedral is the seat of a bishop / 主教座堂 |
| Basilica (宗座圣殿) | Grand church | Specific papal designation / 教宗特别命名 |
| Chapel (小堂) | Small church | Smaller place of worship / 较小的礼拜场所 |
| Orthodox Church (东正教堂) | Greek/Russian church | Use denomination name / 使用教派名称 |
| Mass (弥撒) | Ceremony | Catholic specific term / 天主教特定术语 |
| Eucharist/Communion (圣餐) | Bread and wine ceremony | Sacred sacrament / 神圣圣事 |
| Denomination (教派) | Sect | "Sect" has negative connotations / "教派"而非"邪派" |

**Sacred Text Handling / 经典处理:**
- The Bible should be referred to with its full name or specific book. / 应使用全名或具体书卷名引用圣经。
- Different denominations use different Bible translations — specify which. / 不同教派使用不同圣经译本——需注明。
- The Apocrypha/Deuterocanonical books are accepted by Catholics and Orthodox but not most Protestants. / 次经被天主教和东正教接受，但多数新教不接受。

**Image Guidelines / 图像指南:**
- The Cross is sacred — never use as mere decoration. / 十字架是神圣的——不可仅作装饰。
- Icons (圣像) in Orthodox tradition are considered windows to heaven — treat with reverence. / 东正教中的圣像被视为通往天堂的窗口——须恭敬对待。
- Protestant traditions may be uncomfortable with extensive imagery of saints. / 新教传统可能对大量圣徒图像感到不适。
- Depictions of Jesus vary by culture and denomination; be inclusive. / 耶稣的形象因文化和教派而异；应包容呈现。

**Color Symbolism / 色彩象征:**
- Purple (紫色): Advent, Lent — penitence, royalty / 将临期、大斋期——忏悔、王权
- White (白色): Christmas, Easter — purity, resurrection / 圣诞、复活节——纯洁、复活
- Red (红色): Pentecost, martyrdom — Holy Spirit, blood / 圣灵降临、殉道——圣灵、宝血
- Green (绿色): Ordinary time — growth, hope / 常年期——成长、希望
- Blue (蓝色): Mary, Advent (some traditions) / 圣母玛利亚、将临期（部分传统）
- Platform color: `#3B82F6` (royal blue) / 平台色: `#3B82F6`（皇家蓝）

### 3.2 Pilgrimage Etiquette / 朝圣礼仪

**Dress Code / 着装规范:**
- Vatican/St. Peter's: No bare shoulders, no shorts above the knee, no hats inside. / 梵蒂冈/圣彼得大教堂：不可露肩、不可穿膝上短裤、室内不戴帽。
- Orthodox churches: Women should cover their heads; men remove hats. / 东正教堂：女性应蒙头；男性脱帽。
- Protestant churches: Generally more relaxed, but smart casual preferred. / 新教教堂：通常较宽松，但以商务休闲为宜。

**Photography Rules / 拍摄规定:**
- Most cathedrals allow photography without flash. / 大多数大教堂允许无闪光灯拍照。
- NO photography during Mass or services. / 弥撒或礼拜期间禁止拍照。
- Some Orthodox churches prohibit all photography. / 某些东正教堂完全禁止拍照。
- Sistine Chapel: NO photography (strictly enforced). / 西斯廷教堂：禁止拍照（严格执行）。

**Prayer Times / 礼拜时间:**
- Sunday morning services: typically 9:00-12:00 AM. / 主日崇拜：通常上午9:00-12:00。
- Catholic Mass may be daily; check schedules. / 天主教弥撒可能每日举行；查看时间表。
- Vespers/Evensong: typically 5:00-6:00 PM. / 晚祷：通常下午5:00-6:00。

**Offering Customs / 供养习俗:**
- Candle lighting is common in Catholic and Orthodox churches. / 在天主教和东正教堂中点蜡烛很常见。
- Collection/offering plates during services — participation is optional for visitors. / 礼拜中的奉献盘——访客自愿参与。
- Communion is restricted to baptized members in most traditions. / 多数传统中，圣餐仅限受洗成员领受。

**Gender-Specific Rules / 性别相关规定:**
- Most denominations have no gender restrictions for visitors. / 大多数教派对访客无性别限制。
- Some Orthodox monasteries have gender-restricted areas. / 某些东正教修道院有性别限制区域。
- Mount Athos (Greece): NO women allowed (strictly enforced for over 1000 years). / 阿索斯山（希腊）：女性不得进入（严格执行超过1000年）。

**Shoe Removal / 脱鞋规范:**
- NOT required in most Christian churches. / 大多数基督教堂不要求脱鞋。
- Exception: Ethiopian Orthodox churches require shoe removal. / 例外：埃塞俄比亚东正教堂要求脱鞋。

### 3.3 Calendar & Holy Days / 历法与节日

## 11. Indigenous Spirituality / 原住民灵性

### 11.1 Respectful Representation / 尊重表述

**Proper Terminology / 正确术语:**
| Correct / 正确 | Incorrect / 不正确 | Notes / 备注 |
|---|---|---|
| Indigenous spirituality (原住民灵性) | Primitive religion | NEVER use "primitive" / 绝不使用"原始" |
| Traditional/ancestral practices (传统/祖先实践) | Paganism | "Pagan" is derogatory / "异教"是贬义的 |
| Sacred site (圣地) | Mystical place | Avoid exoticizing / 避免异国情调化 |
| Elder (长老) | Medicine man/Witch doctor | Deeply offensive stereotypes / 极度冒犯的刻板印象 |
| Ceremony (仪式) | Ritual/Magic | Avoid connotations of superstition / 避免迷信暗示 |
| Dreamtime/Dreaming (梦时代) | Myth | It is a living spiritual reality, not myth / 它是活的灵性现实，非神话 |
| First Nations/Aboriginal/Indigenous | Natives/Savages | Use community-preferred terms / 使用社区首选术语 |

**CRITICAL PRINCIPLE:** Indigenous spirituality is NOT one religion — it encompasses THOUSANDS of distinct traditions worldwide. Always specify which tradition you are discussing.

**关键原则:** 原住民灵性不是一种宗教——它涵盖全球数千种不同的传统。始终注明讨论的是哪种传统。

**Image Guidelines / 图像指南:**
- NEVER reproduce sacred imagery without explicit community permission. / 未经社区明确许可，绝不复制神圣图像。
- Some Indigenous artwork is "restricted" — not for public viewing. / 某些原住民艺术是"受限的"——不供公开观看。
- Avoid stock photography of Indigenous ceremonies — these are often staged and disrespectful. / 避免使用原住民仪式的库存图片——这些通常是摆拍且不尊重的。
- Do not use dreamcatchers, spirit animals, or similar items as decorative motifs unless from the relevant tradition. / 不可将捕梦网、灵兽或类似物品作为装饰主题，除非来自相关传统。

**Color Symbolism / 色彩象征:**
- Earth tones (大地色): Brown, ochre, terracotta — connection to land / 棕、赭、赤陶——与大地的联系
- Red ochre (红赭): Sacred in Australian Aboriginal traditions / 在澳洲原住民传统中是神圣的
- Turquoise (绿松石色): Sacred in many Native American traditions / 在许多美洲原住民传统中是神圣的
- Platform color: `#78716C` (warm stone) / 平台色: `#78716C`（暖石色）

### 11.2 Pilgrimage Etiquette / 朝圣礼仪

**CRITICAL: Many sacred Indigenous sites are NOT open to the public.** Visiting without permission is trespassing on sacred ground.

**关键：许多神圣的原住民场所不对公众开放。** 未经许可参访是对圣地的侵犯。

**Dress Code / 着装规范:**
- Varies by community. Ask in advance. / 因社区而异。提前询问。
- Dress modestly and respectfully. / 着装端庄恭敬。

**Photography Rules / 拍摄规定:**
- **DO NOT photograph sacred ceremonies, sites, or objects unless explicitly permitted.** / **未经明确许可，不可拍摄神圣仪式、场所或物品。**
- Uluru: Climbing is now BANNED (since 2019); photography at certain spots is restricted. / 乌鲁鲁：攀登已被禁止（2019年起）；某些地点拍照受限。
- Ask before photographing Indigenous people. / 拍摄原住民前须征询。

**Offering Customs / 供养习俗:**
- Varies enormously by tradition. / 因传统而异，差异巨大。
- Tobacco offerings are sacred in many North American Indigenous traditions. / 在许多北美原住民传统中，烟草供品是神圣的。
- Do not take rocks, plants, or soil from sacred sites. / 不可从圣地带走岩石、植物或土壤。

### 11.3 Calendar & Holy Days / 历法与节日

Indigenous calendars are typically tied to natural cycles (solstices, equinoxes, seasonal changes, agricultural cycles). They vary enormously by tradition and location.

原住民历法通常与自然周期（至日、分日、季节变化、农业周期）相关。因传统和地点而差异巨大。

| Event / 事件 | Notes / 备注 |
|---|---|
| Summer/Winter Solstice (夏/冬至) | Significant across many traditions. / 在许多传统中具有重要意义 |
| Equinoxes (春分/秋分) | Agricultural and ceremonial significance. / 农业和仪式意义 |
| Inti Raymi (太阳节) | June 24 — Inca/Andean tradition. / 六月二十四日——印加/安第斯传统 |
| NAIDOC Week | July (Australia) — celebration of Aboriginal culture. / 七月（澳大利亚）——庆祝原住民文化 |
| Powwow Season | Summer (North America) — social and ceremonial gatherings. / 夏季（北美）——社交和仪式集会 |

### 11.4 Language Considerations / 语言注意事项

**Sacred Languages:** Thousands of Indigenous languages worldwide, many endangered. Respect the language of the specific community.

**Key principle:** Many spiritual concepts do not have English/Chinese equivalents and should be described rather than translated.

**Terms to handle carefully:** Dreamtime, Totem (originated from Ojibwe "ototeman"), Spirit Animal (often misappropriated), Medicine Wheel, Sweat Lodge, Vision Quest

### 11.5 Sensitivity Red Flags / 敏感问题

- **Cultural appropriation is the #1 concern.** Do not borrow, remix, or commercialize Indigenous spiritual practices. / **文化挪用是第一关注点。** 不可借用、混合或商业化原住民灵性实践。
- The history of colonialism, genocide, forced conversions, and residential schools is central context. / 殖民主义、种族灭绝、强制皈依和寄宿学校的历史是核心背景。
- Do not present Indigenous peoples as "relics of the past" — these are LIVING traditions. / 不可将原住民呈现为"过去的遗迹"——这些是活的传统。
- "Shamanism" is a specific term from Siberian traditions — do not apply it universally. / "萨满教"是西伯利亚传统的特定术语——不可普遍应用。
- Avoid the "noble savage" trope. / 避免"高贵的野蛮人"套路。
- Ayahuasca tourism and "spiritual tourism" can be deeply harmful to Indigenous communities. / 死藤水旅游和"灵性旅游"可能对原住民社区造成深重伤害。
- Land rights and sacred site protection are ongoing political issues. / 土地权利和圣地保护是持续的政治议题。

**Content Moderation Keywords:** savage, primitive, uncivilized, witch doctor, voodoo, heathen, pagan, jungle people, tribal (used dismissively)

## 13. Cross-Faith Guidelines / 跨信仰通用指南

### 13.1 General Principles / 一般原则

1. **Equal Respect (平等尊重):** Every faith tradition receives the same level of respectful treatment. No tradition is "superior" or "inferior." / 每个信仰传统获得同等程度的尊重。没有传统是"优越"或"低等"的。

2. **Self-Identification (自我认同):** Use the terms each tradition uses for itself. Do not impose external labels. / 使用每个传统用于自身的术语。不要强加外部标签。

3. **No Ranking (不排序):** Never rank faiths by number of followers, age, or "sophistication." On the platform, the 12 faiths appear in a consistent order based on the seed data, not value judgment. / 不要按信徒数量、历史长短或"精密程度"排列信仰。平台上12个信仰以种子数据为基础的一致顺序出现，而非价值判断。

4. **Diversity Within (内部多样性):** Every faith has internal diversity — denominations, schools, regional variations. Avoid presenting any faith as monolithic. / 每个信仰都有内部多样性——教派、学派、地区变体。避免将任何信仰呈现为铁板一块。

5. **Living Traditions (活的传统):** All 12 faiths are alive and evolving. Do not present any as "ancient relics." / 所有12个信仰都是活的、不断发展的。不要将任何信仰呈现为"古老遗迹"。

### 13.2 Interfaith Comparisons / 跨信仰比较

**DO / 可以:**
- Highlight shared values (compassion, justice, peace) / 突出共同价值（慈悲、正义、和平）
- Note historical connections and mutual influences / 注明历史联系和相互影响
- Present parallel practices (meditation, prayer, pilgrimage) / 呈现平行实践（冥想、祈祷、朝圣）

**DO NOT / 不可:**
- Declare one faith "better" or "more logical" / 宣称某一信仰"更好"或"更理性"
- Reduce faiths to comparative charts that oversimplify / 将信仰简化为过度简化的对比表
- Suggest all faiths "teach the same thing" / 暗示所有信仰"教导相同的东西"
- Use one faith's framework to evaluate another / 用一个信仰的框架评价另一个

### 13.3 Historical Conflicts / 历史冲突

Handle these with extreme care:
- Crusades (Christianity vs. Islam)
- Partition of India (Hinduism vs. Islam vs. Sikhism)
- Israeli-Palestinian conflict (Judaism vs. Islam)
- Buddhist-Muslim tensions in Myanmar
- Anti-Semitism in Christian history
- Colonial suppression of Indigenous spirituality
- China-Tibet situation
- Northern Ireland (Catholic vs. Protestant)
- Sunni-Shia divisions

**Platform principle:** We acknowledge history without taking sides. We focus on the spiritual and cultural richness of each tradition.

**平台原则:** 我们承认历史但不选边站。我们关注每个传统的灵性和文化丰富性。

### 13.4 Content Moderation Universal Rules / 内容审核通用规则

1. Zero tolerance for: hate speech, incitement to violence, blasphemy against any faith, proselytizing/conversion pressure. / 零容忍：仇恨言论、煽动暴力、亵渎任何信仰、传教/皈依压力。

2. Flagged content requires human review before publication. / 被标记的内容须经人工审核后方可发布。

3. User-generated content (journals, reviews) must be moderated against these guidelines. / 用户生成内容（日志、评价）须根据这些指南进行审核。

4. AI responses (Xiaohong chat) must be programmed to deflect theological debate and redirect to cultural appreciation. / AI回复（小鸿聊天）须编程为转移神学辩论并引导至文化欣赏。
